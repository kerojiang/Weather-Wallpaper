# Weather-Wallpaper
# 设置壁纸部分使用了gnome快捷键,目前只能在gnome环境下使用
## 天气壁纸，自动根据当前天气生成对应壁纸  
1.图片来源unsplash接口,一言来源一言接口,天气信息来源wttr.in接口  
2.两种展示方式  
  2.1根据当前天气情况，获取对应天气情况的壁纸并合成天气信息，自动设置为壁纸(使用一段时间后发现天气壁纸数量较少，容易出现重复)  
  2.2获取随机壁纸并合成天气信息,自动设置为壁纸
